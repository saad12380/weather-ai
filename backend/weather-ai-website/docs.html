Getting Started

Welcome to the Weather AI documentation. This guide will help you get started with our APIs and SDKs quickly.

Prerequisites

  • A Weather AI account (Sign up for free)
  • API Key from your dashboard
  • Basic knowledge of REST APIs

Quick Start in 3 Steps

1. Get Your API Key

Navigate to the API Settings in your dashboard to generate your API key.

# Your API Key will look like this
WEATHER_AI_API_KEY=wai_sk_live_1234567890abcdef1234567890abcdef
2. Make Your First API Call

Test your connection with a simple weather data request:

// JavaScript Example
fetch('https://api.weatherai.com/v1/weather/current?location=New+York&api_key=YOUR_API_KEY')
  .then(response => response.json())
  .then(data => console.log(data));
3. Integrate with Your App

Use our SDKs for easy integration:

# Install the Python SDK
pip install weatherai-sdk

# Install the JavaScript SDK
npm install weatherai-js

Authentication

All API requests require authentication using your API key. Include it in the request header or as a query parameter.

Using API Key in Headers

GET /v1/weather/current?location=London HTTP/1.1
Host: api.weatherai.com
X-API-Key: wai_sk_live_1234567890abcdef1234567890abcdef
Content-Type: application/json

Using API Key in Query Parameter

curl -X GET \
  "https://api.weatherai.com/v1/weather/current?location=London&api_key=YOUR_API_KEY" \
  -H "Content-Type: application/json"

Authentication Errors

Error Code Description Solution
401 Unauthorized Invalid or missing API key Check your API key and ensure it's included in the request
403 Forbidden API key doesn't have permission for this endpoint Upgrade your plan or request access
429 Too Many Requests Rate limit exceeded Wait before making more requests or upgrade your plan

Weather Data API

Retrieve current weather conditions, temperature, humidity, wind speed, and other meteorological data.

GET /v1/weather/current

Get current weather data for a specific location.

Parameters
Parameter Type Required Description
location string Required City name, coordinates (lat,lon), or ZIP code
units string Optional Units: metric, imperial, or scientific (default: metric)
language string Optional Language code for descriptions (default: en)
Example Request
curl -X GET \
  "https://api.weatherai.com/v1/weather/current?location=Paris&units=metric&language=en" \
  -H "X-API-Key: YOUR_API_KEY"

JavaScript SDK

Official Weather AI JavaScript SDK for browser and Node.js applications.

Installation

# Using npm
npm install weatherai-js

# Using yarn
yarn add weatherai-js

# Using CDN
<script src="https://cdn.weatherai.com/weatherai-js/v1.0.0/weatherai.min.js"></script>

Quick Example

import WeatherAI from 'weatherai-js';

// Initialize with your API key
const weatherAI = new WeatherAI('YOUR_API_KEY');

// Get current weather
async function getWeather() {
  try {
    const weather = await weatherAI.current('New York');
    console.log(`Temperature: ${weather.temperature}°C`);
    console.log(`Condition: ${weather.weather}`);
    
    if (weather.anomaly_detected) {
      console.log(`Anomaly detected with score: ${weather.anomaly_score}`);
    }
  } catch (error) {
    console.error('Error fetching weather:', error);
  }
}

// Detect anomalies
async function detectAnomalies() {
  const anomalies = await weatherAI.anomalies({
    location: 'London',
    days: 7,
    metrics: ['temperature', 'humidity']
  });
  
  anomalies.forEach(anomaly => {
    console.log(`Anomaly at ${anomaly.timestamp}: ${anomaly.description}`);
  });
}

SDK Methods

Method Description Returns
current(location, options) Get current weather CurrentWeather object
forecast(location, days, options) Get weather forecast Forecast array
historical(location, start, end, options) Get historical weather data Historical data array
anomalies(options) Detect weather anomalies Anomaly detection results
alerts(location, options) Get weather alerts Alert array
