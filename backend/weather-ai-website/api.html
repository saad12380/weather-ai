Getting Started

Welcome to the Weather AI API. This guide will help you integrate our weather intelligence platform into your applications.

Quick Start

To start using the Weather AI API, you'll need an API key. Sign up for a free account to get your key.

Your API Key

Keep this key secret. Never share it in client-side code or public repositories.

wa_sk_live_51Mn6p8K7V4r9x2Yt3wZ6bN8mQ0cV2X5
Make Your First Request

Test your API key with a simple request to get current weather data:

bash 
curl -X GET \
  "https://api.weatherai.com/v1/weather/current?city=New York" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"

Replace YOUR_API_KEY with your actual API key from above.

Authentication

All API requests require authentication using a Bearer token in the Authorization header.

Authentication Header

Include your API key in the Authorization header of all requests:

http 
Authorization: Bearer wa_sk_live_51Mn6p8K7V4r9x2Yt3wZ6bN8mQ0cV2X5
Important: Never expose your API keys in client-side code or public repositories. Use environment variables or secure key management services.
API Key Scopes

API keys can have different permission levels based on your subscription plan:

Scope Free Pro Enterprise Description
weather:read Read access to current weather data
forecast:read Access to weather forecasts
anomaly:detect Anomaly detection endpoints
historical:read Access to historical weather data

Rate Limit Headers

All API responses include rate limit headers:

http 
X-RateLimit-Limit: 60
X-RateLimit-Remaining: 45
X-RateLimit-Reset: 1614556800
Retry-After: 60
Tip: When you hit the rate limit, you'll receive a 429 Too Many Requests response. Use the Retry-After header to determine when to retry.

Core Endpoints

Explore the main API endpoints available in the Weather AI platform.

GET
Current Weather
Free Tier
/v1/weather/current?city={city}&units={units}

Retrieve current weather conditions for a specified location.

Parameters
Parameter Type Required Description
city string Required City name (e.g., "New York") or coordinates (lat,lon)
units string Optional Temperature units: "metric" (Celsius) or "imperial" (Fahrenheit)
lang string Optional Language code (e.g., "en", "es", "fr")
Example Request
javascript 
fetch('https://api.weatherai.com/v1/weather/current?city=London&units=metric', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
})
.then(response => response.json())
.then(data => console.log(data));
