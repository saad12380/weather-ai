General Questions

Weather AI is an advanced artificial intelligence platform designed to detect, analyze, and predict weather anomalies with unprecedented accuracy. Our system works by:

  • Collecting real-time data from thousands of global weather stations, satellites, and sensors
  • Processing this data through machine learning algorithms trained on petabytes of historical weather data
  • Identifying patterns and anomalies that traditional weather systems might miss
  • Providing actionable insights through an intuitive dashboard with predictive analytics

The platform combines computer vision for satellite imagery analysis, natural language processing for report generation, and deep learning for pattern recognition to deliver comprehensive weather intelligence.

Last updated: March 15, 2024

Professional Users

Enterprise Users

Account & Billing

Our 30-day free trial includes full access to all features of the Professional tier:

  • Full Platform Access: Complete dashboard with all visualization tools
  • Real-time Monitoring: Access to live data from 50+ global stations
  • Basic Analytics: Anomaly detection and trend analysis for up to 5 locations
  • Export Capabilities: Download reports in PDF, CSV, and JSON formats
  • API Access: Limited to 1,000 requests per day during trial
  • Email Support: Direct access to our support team

No credit card is required to start the trial. At the end of the trial period, you can choose to upgrade to a paid plan or continue with our limited free tier.

Last updated: February 28, 2024

Yes, you have complete flexibility with your subscription:

Plan Changes

You can upgrade or downgrade your plan at any time. When upgrading, you'll be billed immediately for the prorated difference. When downgrading, changes take effect at your next billing cycle.

Cancellation

You can cancel your subscription at any time. After cancellation, you'll retain access until the end of your current billing period. We don't charge cancellation fees.

Annual Plan Bonus: If you cancel an annual plan before the end of your term, we'll refund the unused portion on a prorated basis minus any discounts applied.
Last updated: February 25, 2024

Weather AI is designed to work on modern devices and browsers with minimal requirements:

Web Browser Requirements
  • Chrome: Version 88+
  • Firefox: Version 85+
  • Safari: Version 14+
  • Edge: Version 88+

JavaScript must be enabled. The dashboard uses WebGL for 3D visualizations, which requires hardware acceleration support.

Mobile Applications
  • iOS: Version 14.0+
  • Android: Version 8.0+

Our mobile apps are available on the App Store and Google Play Store. Push notifications require appropriate permissions.

API Integration

For API integration, any programming language with HTTP client support can be used. Our official SDKs support:

  • Python 3.7+
  • Node.js 14+
  • Java 11+
Network Requirements

Minimum bandwidth of 1 Mbps for standard usage, 5 Mbps for high-resolution data streaming. WebSocket connections are used for real-time updates.

Corporate firewalls may need to allow connections to api.weatherai.com (port 443) and ws.weatherai.com (port 443).

Last updated: February 10, 2024
